## 产品方向
**护身小助手** - 久坐健康提醒工具

### 核心功能（已完成）
1. ✅ 定时震动提醒（30/45/60分钟可选）
2. ✅ 三个健康动作：眺望远方 / 颈椎伸展 / 起身走动
3. ✅ 3D人物动画引导
4. ✅ 今日提醒/完成次数统计
5. ✅ 连续打卡激励机制
6. ✅ 成就徽章系统（10个徽章）
7. ✅ 移动端 PWA 优化
8. ✅ 预览动画功能
9. ✅ 已部署到 GitHub Pages

### 待优化方向
- UI 细节打磨
- 动画流畅度优化
- 用户体验细节
- 深色模式支持
- 音效增强
- 数据导出功能

### 产品定位
- 不是运动健身 App（不与 Keep 竞争）
- 不是医疗诊断 App（只提供通用健康建议，无资质风险）
- 核心价值：在用户沉浸学习/游戏时，强制拉他们休息 1 分钟

### 差异化
