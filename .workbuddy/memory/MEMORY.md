## 产品方向
**护身小助手** - 久坐健康提醒工具

### 核心功能
1. 定时震动提醒（30/45/60分钟可选）
2. 三个健康动作：眺望远方 / 颈椎伸展 / 起身走动
3. 今日提醒/完成次数统计
4. 移动端优先设计，支持添加到手机主屏幕

### 产品定位
- 不是运动健身 App（不与 Keep 竞争）
- 不是医疗诊断 App（只提供通用健康建议，无资质风险）
- 核心价值：在用户沉浸学习/游戏时，强制拉他们休息 1 分钟

### 差异化
- 长震动 + 弹窗提醒（现有产品空白）
- 组合式动作建议（眺望+伸展+走动，三合一）
- 简洁无干扰，专注"提醒"这一个功能
